# AnchorTest

A full-stack application with a React frontend and an Express backend, both built with TypeScript.

## Project Structure

- **frontend/**: React app powered by Vite, TypeScript, and Tailwind CSS.
- **backend/**: Express app powered by TypeScript, handling API requests and file uploads.

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Adityachoubey-git/Anchortest.git
   cd Anchortest
   ```

2. Set up the Backend:
   ```bash
   cd backend
   npm install
   # Create a .env file based on your requirements
   npm run dev
   ```

3. Set up the Frontend:
   ```bash
   cd frontend
   npm install
   # Create a .env file based on your requirements
   npm run dev
   ```
4. .env backend
```
PORT=
MONGODB_URI=
JWT_SECRET=
```
